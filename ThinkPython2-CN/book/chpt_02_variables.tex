%ğŸ% \begin{itemize}
%ğŸ%
%ğŸ% \item {\bf P}arentheses have the highest precedence and can be used
%ğŸ% to force an expression to evaluate in the order you want. Since
%ğŸ% expressions in parentheses are evaluated first, {\tt 2 * (3-1)} is 4,
%ğŸ% and {\tt (1+1)**(5-2)} is 8. You can also use parentheses to make an
%ğŸ% expression easier to read, as in {\tt (minute * 100) / 60}, even
%ğŸ% if it doesn't change the result.
%ğŸ%
%ğŸ% \item {\bf E}xponentiation has the next highest precedence, so
%ğŸ% {\tt 1 + 2**3} is 9, not 27, and {\tt 2 * 3**2} is 18, not 36.
%ğŸ%
%ğŸ% \item {\bf M}ultiplication and {\bf D}ivision have higher precedence
%ğŸ%   than {\bf A}ddition and {\bf S}ubtraction.  So {\tt 2*3-1} is 5, not
%ğŸ%   4, and {\tt 6+4/2} is 8, not 5.
%ğŸ%
%ğŸ% \item Operators with the same precedence are evaluated from left to
%ğŸ%   right (except exponentiation).  So in the expression {\tt degrees /
%ğŸ%     2 * pi}, the division happens first and the result is multiplied
%ğŸ%   by {\tt pi}.  To divide by $2 \pi$, you can use parentheses or write
%ğŸ%   {\tt degrees / 2 / pi}.
%ğŸ%
